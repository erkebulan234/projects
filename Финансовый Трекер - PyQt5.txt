# 💰 Финансовый Трекер - PyQt5

Современное приложение для управления личными финансами с красивым iOS-style дизайном.

## 📱 Возможности

### Основные экраны:
1. **Главный экран (Dashboard)** - быстрый доступ ко всем функциям
2. **Затраты** - просмотр расходов по категориям
3. **Подписки** - управление подписками
4. **Добавление расходов** - быстрое добавление новых трат
5. **Статистика** - анализ и графики
6. **Профиль** - настройки пользователя
7. **Бюджет** - планирование месячного бюджета

### Функциональность:
- ✅ Навигация между экранами
- ✅ Анимированные переходы
- ✅ Красивые градиенты
- ✅ Прогресс-бары
- ✅ Кастомные виджеты
- ✅ Адаптивный интерфейс

## 🚀 Быстрый старт

### Установка зависимостей:

```bash
pip install PyQt5
```

### Запуск приложения:

```bash
# Главное приложение с навигацией
python main_app_navigation.py

# Или отдельные экраны:
python finance_tracker_main.py      # Экран затрат
python subscriptions_screen.py      # Подписки
python add_expense_screen.py        # Добавление расхода
python statistics_screen.py         # Статистика
python profile_settings_screen.py   # Профиль
python budget_planning_screen.py    # Бюджет
```

### Демо кастомных виджетов:

```bash
python custom_widgets.py
```

## 🎨 Кастомные виджеты

### 1. AnimatedButton
Кнопка с анимацией нажатия

```python
from custom_widgets import AnimatedButton

btn = AnimatedButton("Нажми меня!")
btn.clicked.connect(my_function)
```

### 2. CircularProgressBar
Круговой прогресс-бар с процентами

```python
from custom_widgets import CircularProgressBar

progress = CircularProgressBar()
progress.setValue(75)  # 75%
```

### 3. GradientCard
Карточка с красивым градиентом

```python
from custom_widgets import GradientCard

card = GradientCard("#667EEA", "#764BA2")
```

### 4. CategoryIcon
Иконка категории с фоном

```python
from custom_widgets import CategoryIcon

icon = CategoryIcon("🍔", "#FF6B6B", size=50)
```

### 5. NotificationBanner
Уведомления с автоматическим скрытием

```python
from custom_widgets import NotificationBanner

banner = NotificationBanner("Расход добавлен!", "success")
```

### 6. AmountInput
Поле ввода с автоформатированием сумм

```python
from custom_widgets import AmountInput

amount = AmountInput()
# Автоматически форматирует: 1000 → 1,000.00
```

### 7. ToggleSwitch
iOS-style переключатель

```python
from custom_widgets import ToggleSwitch

switch = ToggleSwitch()
switch.toggled.connect(on_toggle)
```

### 8. ExpenseCategorySelector
Селектор категорий расходов

```python
from custom_widgets import ExpenseCategorySelector

selector = ExpenseCategorySelector()
selector.categorySelected.connect(on_category_selected)
```

## 🔧 Использование навигации

```python
from main_app_navigation import FinanceApp

app = FinanceApp()

# Переход между экранами
app.navigateTo("expenses")      # Затраты
app.navigateTo("subscriptions") # Подписки
app.navigateTo("add_expense")   # Добавить расход
app.navigateTo("statistics")    # Статистика
app.navigateTo("profile")       # Профиль
app.navigateTo("budget")        # Бюджет
app.navigateTo("main")          # Главная

# Добавление расхода
app.addExpense(
    amount=50.00,
    category="Еда",
    description="Обед в ресторане",
    date="2025-11-26"
)
```

## 🎯 Основные классы

### FinanceApp
Главное приложение с управлением всеми экранами

**Методы:**
- `navigateTo(screen_name)` - навигация между экранами
- `addExpense(amount, category, description, date)` - добавить расход

**Свойства:**
- `expenses` - список всех расходов
- `budget` - месячный бюджет
- `categories` - данные категорий

### MainScreen

### ExpensesScreen

### AddExpenseScreen

### StatisticsScreen

## 📝 Примеры кода

### Создание экрана с навигацией

```python
class MyScreen(QtWidgets.QWidget):
    def __init__(self, parent):
        super().__init__()
        self.parent = parent
        self.setupUI()
        
    def setupUI(self):
        layout = QtWidgets.QVBoxLayout(self)
        
        # Кнопка возврата
        backBtn = QtWidgets.QPushButton("← Назад")
        backBtn.clicked.connect(
            lambda: self.parent.navigateTo("main")
        )
        layout.addWidget(backBtn)
```

### Добавление карточки

card = QtWidgets.QWidget()
card.setFixedHeight(100)
